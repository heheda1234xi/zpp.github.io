Set和Map数据结构

1.Set

Set是ES6新提供的数据结构，类似于数组，但是Set集合里面的值都是唯一的，是不重合的。

向Set集合中添加值的时候不会发生值类型的转换，例如5和’5’是两种不相等的值。

Set实例的属性和方法

Set实例的属性：

  • Set.prototype.constructor:构造函数，默认是Set()函数。
  • Set.prototype.size:获取Set集合中元素的个数

Set实例的方法：

  • add(value):向Set实例中添加元素值，最后会返回Set实例本身。
  • delete(value):删除Set实例中的元素值，最后返回Set实例本身。
  • has(value):返回一个布尔值，表示Set集合中是否存在指定的值，true表示存在，false表示不存在。
  • clear():表示清除Set实例中所有的数据值，清空Set集合。

    2.WeakSet

WeakSet与Set类似，集合里面的值都是不能重复的，区别是WeakSet里面的元素只能是对象，不能是其他类型的元素。
WeakSet()函数是一个构造函数，可以用于创建WeakSet实例，程序实例：

WeakSet实例的操作方法：

  • WeakSet.prototype.add(value):向WeakSet实例添加一个新成员。
  • WeakSet.prototype.delete(value):从WeakSet中删除一个成员。
  • WeakSet.prototype.has(value):返回一个布尔值，用于判断WeakSet中是否有指定的值。

WeakSet没有size属性所以没有办法遍历它的成员。WeakSt的键指向的对象不会计入垃圾回收机制。

3.Map

Map数据结构中保存的是键值对的集合，跟对象不同的是任何类型的值都可以作为键，包括对象也可以当作键。

Map实例的属性和操作方法：

  1. size:返回Map集合中元素的个数，程序示例：
  2. set(key, value)方法用于向Map集合中添加新的键值对。该方法返回的是添加新元素之后的整个Map结构。如果key已经有值了，就更新key对应的值，如果没有key就创建一个键值对添加在Map集合里面。程序示例：
  3. get(key)表示从Map结构中获取key对应的值。
  4. has(key)返回一个布尔值，表示Map实例中是否包含指定的值，true表示包含，false表示不包含。
  5. delete(key)表示删除Map实例中指定的键值对。
  6. clear()表示清空Map实例集合。

Map实例的遍历方法

  • keys():返回键名。
  • values():返回键值。
  • entries():返回键值对。
  • forEach():遍历所有的Map成员。

4.WeakMap

WeakMap与Map相似，也是存储键值对的集合，但是WeakMap只接受对象作为键名称，不接受其他类型的键。WeakMap键指向的对象不会被计入垃圾回收机制。
